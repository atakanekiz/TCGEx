This website was built using RShiny.

Some of the landing page visuals were obtained from pixabay

The pre-processed/uploaded data and the analysis code is executed on GNU/Linux virtual servers accessed via a secure HTTPS connection on IzTech campus. User data is stored as a temp file for the duration of the session, and it is deleted after the session is terminated. However, we do not recommend analyzing identifiable patient information on the platform. For such analysis needs, one can download the TCGEx pipeline from GitHub or Docker and run a local instance

TCGEx was developed by the Ekiz Lab in the Department of Molecular Biology and Genetics at Izmir Institute of Technology, Turkey
